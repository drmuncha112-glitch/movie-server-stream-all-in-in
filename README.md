# Movie Server Stream All In

Torrent streaming server with instant playback and Jellyfin integration.

This server lets you:

* Stream torrents instantly
* Generate STRM files
* Connect streams to Jellyfin
* Avoid storing full downloads

---

## 🚀 Quick Start (2 minutes)

### 1. Install Node.js

Download Node.js from https://nodejs.org

### 2. Clone the repository

git clone https://github.com/drmuncha112-glitch/movie-server-stream-all-in.git

### 3. Enter the folder

cd movie-server-stream-all-in

### 4. Install dependencies

npm install

### 5. Start the server

node server.js

---

## 🌐 Open the server

After starting the server, open:

http://localhost:9000

---

* Instant torrent streaming
